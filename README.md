# 工具类库
### 这个类库中包含常用的工具类，例如日志、sp、网络连接、sd卡、屏幕大小、软键盘等。
### 你可以通过以下方式引入到自己的项目中
### 将以下代码加入到你的项目根下的build.gradle中
<pre>allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}</pre>
### 然后再添加依赖
<pre>dependencies {
	        compile 'com.github.codeteenager:UtilsLib:v1.0'
	}</pre>
接下来你就可以使用了。
